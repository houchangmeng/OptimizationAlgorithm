# OptimizationAlgorithm

The most common optimization algorithm, implemented by Eigen.

## Requirement

python: matplotlib  
c++: Eigen

## Run

```{bash}
git clone git@github.com:houchangmeng/OptimizationAlgorithm.git
cd OptimizationAlgorithm/
build.bash
./bin/example 
```
